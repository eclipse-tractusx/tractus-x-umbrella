# Legacy per-OS documentation (archived)

The files in this folder are the **byte-identical duplicates** that previously
lived under `docs/user/{linux,mac,windows}/{installation,network,setup}/`.

They were archived during the Release 25.12 documentation cleanup:

- The installation procedure is now consolidated in
  [`docs/user/common/installation.md`](../../user/common/installation.md).
- The TLS reference is at
  [`docs/user/common/network/tls.md`](../../user/common/network/tls.md).
- The Semantic Hub setup is at
  [`docs/user/common/setup/semantic-hub.md`](../../user/common/setup/semantic-hub.md).
- OS-specific cluster + DNS setup remains in each
  [`docs/user/{linux,mac,windows}/README.md`](../../user/).

Nothing here is referenced from the live documentation. Kept for traceability.
